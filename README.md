# Next.js Redux Template

Next.js and React/Redux code template with Firebase Authentication

## Config

src/configs/firebase.ts

```
const config = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: ""
};
```

## Run

```
$ yarn
$ yarn run dev
```
